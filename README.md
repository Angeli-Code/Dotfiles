# Dotfiles
A collection my personal configuration files

### Configuration File Contents
tmux.conf: config for Tmux. A terminal multiplexor. I use it for session management.
vimrc: config for Vim. Pretty minimal with no LSP but it keeps me from getting distracted.
bashrc: config for my bash shell. Pretty minimal. Only added vim motions and a custom PS1
config: config for i3 settings. Removes borders and starts other processes like polybar and a custom background
config.ini:c config for polybar. A customized taskbar for i3.
touchscreen.conf: config file that disables my laptops touch screen. 
alacritty.toml: config for my terminal emulator.

### Distrobutions
Fedora has always been super stable for me. I have used Ubuntu, Arch, Debian, Manjaro, and PopOS. Fedora has 
provided a more modern but stable experience and I plan on staying here for a long time. 

### Scripts
- fedora-install-script: Automated installation of packages and file sorting
automation of how I like my setup.
- obsidian: automates opening obsidian app image. Allowing it to be opened through dmenu
